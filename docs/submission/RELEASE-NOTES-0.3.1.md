# Mermaid Iconify 0.3.1

Initial submission-ready release.

## Features

- discover Iconify icon packs from the official catalog
- enable and disable packs from plugin settings
- register selected packs in Mermaid dynamically
- browse and search icons inside a pack
- preview icons visually in the picker
- copy `prefix:name` tokens for Mermaid usage
- copy Mermaid snippet examples

## Notes

This plugin uses external Iconify endpoints for catalog discovery, icon lookup, SVG previews, and icon pack loading.
No vault content or note content is transmitted.
